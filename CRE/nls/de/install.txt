2. Installationsanweisungen
===========================
Um die Clients für UNIX zu installieren, führen Sie das Skript
setupwfc aus und folgen Sie den Anweisungen auf dem Bildschirm. Das
Standard-Installationsverzeichnis ist /usr/lib/ICAClient für Installationen
vom Superuser und sonst $HOME/ICAClient/$OS. Diese Angabe kann jedoch 
während der Installation geändert werden. 

Wird das Standard-Installationsverzeichnis nicht verwendet, muss die
Umgebungsvariable ICAROOT so gesetzt werden, dass sie auf das
tatsächliche Installationsverzeichnis verweist. 

Mit dem Skript setupwfc können die Clients für UNIX auch 
deinstalliert werden.
